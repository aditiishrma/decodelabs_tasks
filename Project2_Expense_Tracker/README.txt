EXPENSE TRACKER SYSTEM

Developed For:
DecodeLabs Internship – Project 2

Description:
This project is a Python-based Expense Tracker System that helps users record and manage expenses.

Features:

2. View Expenses
3. Search Expense
4. Delete Expense
5. Total Expense Calculation
6. Expense Statistics
7. Category Summary
8. Automatic File Saving

- Dictionaries
- Loops
- File Handling
- Exception Handling
- Input Validation

Files:
